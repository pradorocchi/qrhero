## Implementation

### Install

You can get MarkdownHero with **CocoaPods**.

Add to your Podfile

```
pod 'QRhero'
```

Install and download

```
> pod install
```

If you are wondering what is CocoaPods take a look at: [https://cocoapods.org](https://cocoapods.org)

### Usage

Import MarkdownHero in your class

```
import QRhero
```
